# 🚗 ParkCode

A crowdsourced smart parking web application that helps users find and share real-time parking availability using map-based visualization and Plus Code precision.

---

## 🧠 Problem
Finding parking in busy areas is time-consuming and inefficient.  
Existing solutions lack real-time updates and precise location data.

---

## 💡 Solution
ParkCode allows users to:
- 📍 Find nearby parking spots  
- 📢 Report available parking  
- 🔄 Update parking status in real time  
- 🔑 Share precise locations using Plus Codes  

---

## 🚀 Features (MVP)
- User authentication  
- Map-based parking view  
- Search using Plus Code / location  
- Report parking spots  
- Confirm parking availability  
- Expiry-based data validation  

---

## 🛠️ Tech Stack
- Frontend: React  
- Backend: Node.js + Express  
- Database: MongoDB  
- Maps: Google Maps API  

---

## 🔄 How it Works
1. User logs in  
2. Searches location (Plus Code / GPS)  
3. Views nearby parking spots  
4. Reports or updates parking status
